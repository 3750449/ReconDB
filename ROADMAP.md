# ReconDB Roadmap

ReconDB is being developed as a full-stack cybersecurity asset intelligence platform for authorized lab systems, personal infrastructure, and security learning environments.

---

## Completed

- MySQL relational database schema
- Seed data for sample assets
- Nmap XML ingestion
- Python ingestion scripts
- DNS enrichment
- WHOIS enrichment
- SQL reports
- Bash pipeline automation
- Custom target scanning
- Custom port scanning
- Reset and no-reset scan modes
- Scan job history logging
- FastAPI backend
- Swagger API documentation
- React dashboard
- Dashboard summary cards
- Targets table
- Open ports table
- Domains table
- WHOIS table
- Scan history table
- Global dashboard search
- No-results messages
- Project README
- Setup guide
- Changelog
- Architecture documentation
- Version tags
